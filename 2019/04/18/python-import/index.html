__import__详解

发表于 2019-04-18 | 分类于 Python

  1. __import__函数的作用

    • 使用import导入python模块时,默认调用的是__import__函数.
    • 直接使用该函数,一般用于动态加载模块.

  2. __import__函数的定义

    在__builtin__.py中可以看到__import__函数的定义如下:
    __import__(name, globals={}, locals={}, fromlist=[], level=-1)

  3. __import__函数的参数

    由__import__函数的定义可以看出, 该函数一共有五个参数, 其中name是必选参数, 其它是可选参数.
    特别要注意:

    • 只传入使用name参数,只能导入最顶层模块(如示例中In[2]和Out[2]);
    • 同时传入name和fromlist参数(只要list的len不是0即可,根本不care传入的内容),则可以导入子模块(如示例中In[3]和Out[3]);

    • level参数,指定导入方式（相对导入或者绝对导入,默认两者都支持）
